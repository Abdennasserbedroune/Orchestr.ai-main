# n8n workflows (vendored)

This folder is a full local copy of workflows from [Zie619/n8n-workflows](https://github.com/Zie619/n8n-workflows).

- Raw JSON files: `workflows/n8n/workflows/**.json`
- Metadata catalog:
  - `workflows/n8n/catalog/index.json`
  - `workflows/n8n/catalog/by-category/*.json`

To update the snapshot, run the GitHub Action **Sync n8n workflows** (workflow_dispatch).
